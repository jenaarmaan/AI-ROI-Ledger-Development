# AI ROI Ledger

**Day 2 / 50 ‚Äì AI Systems Engineering Challenge**

AI systems often become cost centers because their impact is not measured against
business outcomes. This project demonstrates a simple but powerful idea:

> **Treat AI usage like financial transactions and compute ROI per use case.**

- Weekly-style ROI summary to identify:
  - High-value AI workflows
  - Loss-making AI usage
  - Areas needing more data
